# SME AI Readiness Assessment Tool

A web-based AI assessment platform for SME owners. They complete a 5-minute form and receive a professionally designed, consultant-level PDF report via email.

## Quick Start

```bash
make install
make dev
```

Backend: `http://localhost:3001` | Frontend: `http://localhost:3000`

## Deployment

See `docker-compose.yml` for Coolify deployment.
